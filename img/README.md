Imagens do mapa — instruções

Arquivos placeholder criados anteriormente (retirados agora). Se preferir usar imagens reais, coloque na pasta `img/` com qualquer nome e atualize os `src` em `index.html` e nas páginas de destino.

Sugestões para substituir:
- Use imagens JPG/PNG com boa resolução (recomendo 1200×600 para `map-overview`, 900×500 para os recortes, e 400×220 para as miniaturas).
- Nomes dos arquivos devem ser exatamente os listados acima para que o site aponte automaticamente.
- Para melhorar contraste das anotações (círculos/etiquetas), prefira imagens noturnas/sem excesso de elementos no topo.

Se quiser, eu posso:
- Subir os anexos que você enviou e gerar automaticamente os recortes (map-matriz, map-deposito, map-balanca) usando esses arquivos.
- Ajustar cores e textos das anotações conforme identidade Riofer.

Confirme se deseja que eu importe os anexos e execute os recortes agora (opção recomendada).
